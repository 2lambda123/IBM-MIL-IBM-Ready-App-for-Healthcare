Converts the string representation of a number (both integer and floating point values) to a string containing the number as an integer with the necessary number of leading zeros to match the desired width.

Examples:

  • number: 3.14, width: 3 -> 003
  • number: 3, width: 2 -> 03
  • number: 3.14159, width: 1 -> 3
