Class: choreographerService

ReadyAppHC. choreographerService

new choreographerService()

An Angular service that handles the creation of a so-called delta value that is used by graph directives (bar-graph, line-graph, and time-graph) to determine the appropriate range for the x-axis, depending on the selected tab. In other words, this service creates a number called delta that is used by graph directives to space the x-values out so that graphs with more data are more crowded and graphs with less data are less crowded.
Properties:
Name Type Description
delta integer The delta value that defines the appropriate range for the x-axis, depending on the selected tab.
