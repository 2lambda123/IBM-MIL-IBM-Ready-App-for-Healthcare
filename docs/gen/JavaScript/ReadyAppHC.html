ReadyAppHC

Physio is dependent on the following JavaScript libraries: Angular, D3, and Moment. Angular handles the overarching structure of the hybrid parts of Physio, D3 handles SVG manipulation, and Moment helps us parse and display dates at certain points in the application. From this point onward, this documention will assume that you understand all three of these JavaScript libraries fairly well, as well as understand the standard file structure used with these libraries. All JavaScript files live under the /html/js/src directory, in which files are broken down appropriately (controllers, directives, etc). We also use Sass to compile our extended CSS. All Sass files live under the /html/scss directory, in which files are broken down appropriately (partials, modules, etc). Like tradiational web development, all imports are done in /html/index.html. You will need to run the following Sass command in order to write in Sass: sass --style=compressed --sourcemap=none -C --watch scss/main.scss:css/style.css. If you are developing the hybrid portions of Physio in a browser (like Chrome, Safari, etc), you will need to run Node with this command: node test_server.
Defines the ReadyAppHC module, as well as sets up routing for the Angular app.
