Class: milCtrl

ReadyAppHC. milCtrl

new milCtrl()

The basis for every controller in Physio. Think of this as Object in Java or NSObject in Cocoa. milCtrl provides important functions that all controllers in Physio must possess, including the ability to route to other controllers in Physio. It is highly recommended that you extend this controller when implementing a new controller class. Creating a controller that is not a decendent of this class is not necessarily an error, but makes your life harder in the long run.
