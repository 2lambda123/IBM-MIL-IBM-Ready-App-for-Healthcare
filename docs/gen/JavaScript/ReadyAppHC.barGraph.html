Class: barGraph

ReadyAppHC. barGraph

new barGraph()

A directive that defines a bar graph in Physio. This directive must exist in a descendent of milSyncedGraphCtrl in order to function correctly. This graph is also decorated with three attributes: consume, a reference to a function that returns JSON data that this graph needs to display, transform, a reference to the shared transformation integer that should be applied to the x-axis on all graphs within the parent controller, and unit, a string that describes the units that this graph is displaying in (ex: bpm, lbs, kcal, etc.).
